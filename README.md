# SignalLab

SignalLab is an interactive dashboard for signal analysis, built with Python and Streamlit. It's designed as a hands-on tool for the "Analisi dei Segnali" course.

## Features
*   Perform and visualize signal convolution step-by-step.

## Getting Started

### Prerequisites

*   Python 3.8+
*   pip

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/cescolamarca/signallab.git
    cd signallab
    ```

2.  Install the required packages:
    ```bash
    pip install -r requirements.txt
    ```

### Running the Dashboard

To start the dashboard, run the following command:

```bash
streamlit run app.py
```
